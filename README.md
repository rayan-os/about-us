# Passage About Us Page

A modern, minimal About Us page design for Passage.

## Getting Started

### Prerequisites
- Node.js 18+ 

### Installation

```bash
# Clone the repo
git clone https://github.com/rayan-os/about-us.git
cd about-us

# Install dependencies
npm install

# Run locally
npm run dev
```

### View the page

Open [http://localhost:3000/about-new](http://localhost:3000/about-new) in your browser.

## Features

- Dark/light theme toggle
- Responsive design
- Team photo section
- Investor logos
- Animated grid lines
- Minimal, Vercel-inspired aesthetic
